# Privacy policy template

This is a starting template only. Get legal review before publishing.

## What the extension does

The extension reads text only when the user explicitly asks it to read the currently focused editable field. It sends that text to the PromptPilot backend so the text can be rewritten and returned to the user for review.

The extension does not automatically submit prompts, does not read every page by default, and does not monitor browsing activity in the background.

## Data collected

We collect:

- account email address
- password hash and authentication records
- prompt text that the user explicitly chooses to enhance
- basic usage records such as timestamp, selected mode, and prompt length

## How data is used

We use the data to:

- authenticate users
- enforce usage limits
- provide the prompt enhancement service
- monitor service health and abuse

## Data retention

Prompt text should be retained only as long as needed to operate the service. For this starter, you should decide and document a retention period before launch. The safest default is not to keep raw prompts longer than necessary for processing and debugging.

## Data sharing

We do not sell user data. Prompt text may be sent to our model provider solely to generate the requested rewrite.

## User choices

Users may choose not to use the extension, not to submit a prompt, or to avoid including sensitive information in prompts. Users may request account deletion.
